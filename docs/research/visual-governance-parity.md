**Status:** experiment not yet executed

## Question

Does the full ZeroModel artifact chain answer governance and replay questions materially better than a lightweight baseline consisting of:

- provider or model SHA-256
- preprocessing SHA-256
- calibration SHA-256
- policy SHA-256
- source-scope identity
- Git commit
- append-only JSONL decision logs

## Current evidence

The repository now preserves stronger machine-readable calibration and evaluation artifacts for System B v2.

That is necessary preparation, but it is not yet a completed governance-parity result.

## Current claim boundary

`the ingredients required for a governance-parity experiment are clearer`

`ZeroModel is materially better than a conventional digest-plus-log baseline`
