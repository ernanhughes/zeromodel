# Stage P9 — Temporal Inference Comparison

P9 fits the same deterministic ridge translator family over P8 temporal montage fields and compares it with the existing single-frame translator on aligned held-out interactions.

The comparison reports overall accuracy, accepted-only accuracy, coverage, mean margin, and performance on exact current-frame conflict groups. Rejection uses an explicit top-two margin threshold supplied by the caller.

This stage does not claim that temporal context is causal, that one window length is generally sufficient, or that improved held-out performance transfers beyond the evaluated split.
