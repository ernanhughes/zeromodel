# ZeroModel website

This is the first-principles ZeroModel website.

It is deliberately a zero-dependency static site:

- no framework;
- no build step;
- no generated marketing copy;
- one interactive demonstration implemented with Canvas;
- deployable directly with GitHub Pages.

Run it locally from the repository root:

```bash
python -m http.server 4173 -d site
```

Then open `http://localhost:4173`.

## Website contract

The site must distinguish three kinds of statement:

- **Defined** — part of the VPM artifact specification;
- **Measured** — backed by a reproducible benchmark in this repository;
- **Hypothesis** — proposed research that has not yet been demonstrated.

No performance number should appear on the site unless its benchmark, hardware, dataset, raw output, and comparison baseline are available in `benchmarks/`.
